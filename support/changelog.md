# N.O.V.A - Changelog
Registro de cambios y mejoras de la plataforma

---

## [1.0.0] - 2026-03-20
### Added
- Lanzamiento inicial de N.O.V.A Backend (FastAPI, SQLAlchemy, Redis, MongoDB).
- Frontend móvil con React Native + Expo.
- CI/CD premium con GitHub Actions y GitLab CI.
- Infraestructura desplegada con Terraform, Kubernetes y Ansible.
- Integración de pagos (Stripe, PayPal, Google Play).
- Observabilidad con Prometheus, OpenTelemetry y Loguru.

### Fixed
- Corrección de validación de tokens en el flujo de autenticación.
- Ajustes en CORS y middleware de seguridad.

### Changed
- Optimización de Dockerfiles con multi-stage builds.
- Configuración premium de `requirements.txt` y `pyproject.toml`.
