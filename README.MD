# 🚀 N.O.V.A - Plataforma de Inteligencia Artificial

Bienvenido a **N.O.V.A**, una plataforma de Inteligencia Artificial con agentes especializados (Estudiante, Programador, Secretario, Inversor, Creativo).  
Este proyecto integra **backend en FastAPI**, **frontend móvil en React Native/Expo**, y despliegue en **infraestructura cloud con Docker, Kubernetes, Terraform y CI/CD premium**.

---

## ✨ Características principales
- **Backend (FastAPI)**: APIs seguras y escalables con SQLAlchemy, Redis y MongoDB.
- **Frontend (React Native + Expo)**: aplicación móvil con soporte multilenguaje (i18n).
- **Autenticación**: JWT, hashing seguro con bcrypt y passlib.
- **Observabilidad**: Prometheus, OpenTelemetry, Loguru.
- **Pagos**: integración con Stripe, PayPal y Google Play.
- **Infraestructura**: Terraform, Kubernetes (EKS), Docker Compose, Ansible.
- **CI/CD**: pipelines premium en GitHub Actions y GitLab CI.
- **Seguridad**: auditoría, CORS, trazabilidad y alertas en tiempo real.
